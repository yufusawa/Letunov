# Инструкция
# по работе с git для студентов МФЮА - МФЮИ - МАСИ
- открыть браузер (Chrome желательно) и зайти на сервис git: ```github.com``` ```gitflic.ru``` и тд...
- сделать этот браузер браузером по умолчанию
- ввести логин и пароль
- создать публичный проект с ```README``` файлом
- скопировать ссылку на проект ```http://.....```
- открыть папку ```"Документы"```
- Создать папку ```"git"```
- Открыть VSCode
- ```"File"``` - ```"Open foder"``` выбрать папку ```Документы/git```
- ```"Terminal"``` - ```"New Terminal"```
- В Терминале внизу пишем 

```git clone http://...``` где http://... - это скопрированна ссылка на проект. "Enter"

- если возникла ошибка и много крсного цвета, то необходимо установить Git Bash. Для этого в браузере в поисковой строке введите git bash и по первой ссылке скачайте и установите по умолчанию по всем пунктам.

- Затем нужно войти в каталог с только что склонированным репозиторием 

```cd ...```
 где ... - это имя каталога с репозиторием
- затем добавляем в каталог необзодимые файлы, пишем README...

```git status```
- здесь отображаются еще не отслеживаемые файлы "красным цветом"

``` git add . ```

```git status```

- после этой команды файлы должны стать "зеленым цветом"

```git commit -m 'add your comment'```

```git status```

```git push```

- откроется браузер и нужно нажать зеленую кнопку 

- обновляем страницу с репозиторием и проверяем наличие файлов на сервере
